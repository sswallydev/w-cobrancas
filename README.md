# W Cobranças

Sistema de leitura de boletos por OCR e envio automático de cobranças por e-mail.

## Funcionalidades

- Upload de boletos em PDF ou imagem
- Leitura OCR com Tesseract
- Conversão de PDF com Poppler
- Extração de nome, valor e vencimento
- Correção manual dos dados
- Prévia do e-mail antes do envio
- Envio automático por SMTP
- Histórico de cobranças
- Atualização de status

## Tecnologias

- Python
- FastAPI
- MySQL
- SQLAlchemy
- Pydantic
- HTML, CSS e JavaScript
- Tesseract OCR
- Poppler
- SMTP

## Como executar

```bash
pip install -r requirements.txt
python run.py
